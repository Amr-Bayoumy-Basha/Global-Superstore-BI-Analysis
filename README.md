## Technical Implementation

This project leverages advanced Power BI capabilities and sophisticated analytical techniques:

### Core Technical Components

| Component | Implementation |
| :--- | :--- |
| **Data Architecture** | Star Schema with optimized Fact and Dimension tables |
| **Advanced DAX Patterns** | WINDOW(), RANKX(), time intelligence, disconnected tables |
| **Customer Analytics** | Cohort analysis, retention tracking, CLV calculations |
| **Product Analytics** | Composite scoring (weighted profit, sales, returns) |
| **Dynamic Analysis** | Pareto analysis with user-selectable metrics |
| **Time Intelligence** | YTD, rolling averages, period-over-period comparisons |

### Key Technical Achievements

- **Dynamic Pareto Analysis**: Fully interactive Pareto curves using WINDOW() function with disconnected X_axis table for precise cumulative calculations across any selected metric
- **Customer Lifetime Value (CLV)**: Comprehensive calculation incorporating Average Purchase Value, Purchase Frequency Rate, and Customer Lifespan based on churn analytics
- **Cohort Retention Tracking**: Year-over-year retention analysis for customer cohorts using advanced filtering and calculated columns
- **Composite Product Ranking**: Multi-dimensional scoring system weighting profit (50%), sales (30%), and return rate (20%) to identify true performers
- **Conditional Time Intelligence**: Rolling averages with data boundary validation to prevent distorted calculations at dataset edges
